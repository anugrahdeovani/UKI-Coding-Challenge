# Installation How To Run
### Deovani Anugrah Alhafid (anugrahdeovani@gmail.com / deovani.alhafid@xsis.co.id)

Install the dep and devDep and start server side and client side.

#### Server
```sh
$ cd server
$ npm install
$ npm start
```

#### Client
```sh
$ cd client
$ npm install
$ npm start
```

### DB config on Server
```sh
{
  "local": {
    "host": "mongodb://localhost",
    "port": 27017,
    "name": "employee_db"
  }
}
```
